pip install PySide6

วิธีการใช้งาน
เพิ่มงานใหม่ กดปุ่มมุมบนขวา
ใส่ชื่องาน, วันครบกำหนด, และระดับความสำคัญ (High / Medium / Low)
กดบันทึก งานจะปรากฏในรายการทันที

มาร์คงานว่าเสร็จแล้ว
คลิกเลือกงานในรายการ
กดปุ่ม เสร็จแล้ว งานจะแสดงแบบขีดฆ่า

แก้ไขงาน
คลิกเลือกงานที่ต้องการแก้
กดปุ่ม แก้ไข
แก้ข้อมูลแล้วกด บันทึก

ลบงาน
คลิกเลือกงานที่ต้องการลบ
กดปุ่มลบ

ค้นหาและกรองงาน
พิมพ์ชื่องานในช่อง ค้นหา เพื่อกรองแบบ Realtime
เลือก Priority จาก Dropdown เพื่อดูเฉพาะงานระดับนั้น
ติ๊ก แสดงที่เสร็จแล้ว เพื่อซ่อน/แสดงงานที่เสร็จไปแล้ว

การแจ้งเตือน
โปรแกรมจะแจ้งเตือนอัตโนมัติเมื่อเปิดขึ้นมา หากมีงานที่ เกินกำหนด หรือ ครบกำหนดวันนี้
